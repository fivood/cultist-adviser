## 1. 核心循环

- 三大基础属性卡：**健康 Health**（体力活）、**理性 Reason**（文书/研究）、**激情 Passion**（绘画/仪式）。初始各 1 张，技能升级可增加（上限 5 张）。
- **时光 Time** verb 每 60 秒自动消耗 1 资金维持生活；⚙ 没有资金时扣 1 健康变成"饥饿"，需入梦+1资金在时限内恢复，否则健康永久损失。
- 大方向：前期稳经济+升技能+买书学语言 → 中期建教团、攒手下/秘传/囚徒 → 后期漫宿探索、冲飞升（欲望 7 级 / 野心 6 级 + 高级秘传仪式）。

## 2. 时节牌库（Time 每轮抽一张，抽空重洗）

牌库构成（共 11 张，wiki 确认）：

| 时节 | 张数 | 效果 | 对策 |
|---|---|---|---|
| 疾病 Sickness | 1 | 扣健康→病痛(倒计时) | 入梦+1资金或1活力治疗，超时转**衰老**（永久损血） |
| 疑心 Suspicion | 2 | 吸秘氛/邪名生成猎人；已有猎人则产证据 | 平时清声名卡；无声名时反而+满足 |
| 绝望 Despair | 2 | 吸走场上所有恐惧 | 备满足；3 张恐惧被吸满→死亡倒计时 |
| 幻象 Visions | 2 | 吸走场上所有痴迷 | 备恐惧或一瞬追忆；3 张吸满→发疯结局 |
| 野心 Ambition | 2 | 吸走欲望卡考验升级 | 3 级后需囚徒/尸体/资金；不满足则产生躁动 |
| 静寂 Silence | 1 | 60 秒内不触发疑心 | 安全窗口，适合做产声名的事 |
| 情愫 Ardours | 1 | 爱人事件，索要资源 | 满足需求或断绝关系 |

⚙ **时节是周期性威胁：绝望/幻象各 2 张意味着平均每轮牌库会被吸 2 次恐惧/痴迷，场上恐惧≥2 就该提前准备对策卡。**

## 3. 三大死亡螺旋（最高优先级警报）

| 螺旋 | 触发 | 对策卡 | 结局 |
|---|---|---|---|
| **绝望 despair**（Bleak Thoughts） | 累积吸满 3 张恐惧 Dread → 倒计时 | 放入**满足 Contentment** | No More（自尽） |
| **幻象 visions**（A Trembling in the Air） | 累积吸满 3 张痴迷 Fascination → 倒计时 | 放入**恐惧 Dread** 或 **一瞬追忆 Fleeting Reminiscence** | Glory（发疯"飞升"） |
| **饥饿 starvation** | 无资金时健康转饥饿 | 入梦+1资金 | 健康耗尽死亡 |

⚙ 关键数值：恐惧、痴迷卡自身 **180 秒**衰变；满足衰变很快，拿到就尽快用。
⚙ 入梦配对消除：恐惧+满足 → 消；痴迷+恐惧 → 得一瞬追忆（更安全的幻象对策，因为恐惧可能被绝望时节吸走）。
⚙ 推荐规则：场上恐惧≥2 且无满足 → 紧急；痴迷≥2 且无恐惧/一瞬追忆 → 紧急。

## 4. 健康状态链

```
疾病时节/工作失败 → 病痛(180s) → 未治疗 → 衰老(永久-1健康)
躁动 Restlessness(60s) → 未处理 → 恐惧 Dread
```
⚙ 病痛、躁动都是**带倒计时的中间状态卡**，出现即应提示：病痛→入梦+资金/活力；躁动→绘画消耗或投入激情。

## 5. 经济

- 每 60 秒固定支出 1 资金；建议**保持 ≥5 资金缓冲**（攻略2建议 20）。⚙ 资金<3 时提示赚钱。
- 收入路线：
  - Glover & Glover 文书职级：Junior(60s/1金) → Middle(60s/1金+加班2金，投理性升) → Senior(70s/3金，需激情) → Board(50s/4金，但自动吸恶名，两次被发现即开除)。
  - 体力活：健康+健康技能 → 1资金+1活力；无技能 30% 受伤，1 级技能 10%。
  - 医师开局：研究所工作 60s/2金，最稳。
  - 绘画：激情→灵感/恶名换钱，会产声名卡。
  - 委托（赞助人）：秘传+理性/激情写论文 → Spintria 古币 → 拍卖行兑现。
- 拍卖行卖物价：等级1→1金、2→2、3→4、4→8、5→16、6→20（部分有额外浮动）。
- 莫兰书店：1 资金随机一本书，库存有限；**买空后获得书店房间（图书馆总部，最优总部）**。

## 6. 技能与碎片（skill fragments）

⚙ 短命碎片（180s 内衰变）：**活力 vitality**（健康系）、**博闻/进阶 erudition**（理性系）、**灵感/微光 glimmering**（激情系）。
- 升级链（攻略2数值）：2 碎片→1 级技能卡；4 碎片+1级技能→2级；6 碎片+2级技能→3级；8 碎片+3级→4级。
- ⚙ 场上凑齐 2 张同类碎片就该尽快研读合成，碎片快过期时紧急提示。（advisor 已实现 STUDY_PAIRS）
- 技能优先级：健康优先升（初始只有 1 张，疾病时节风险大），三技能都到 3 级为中期目标。

## 7. 秘传（Lore）升级

- 同类合成：同属性同等级 ×2 + 理性/博闻检定 → 高 2 级秘传。检定用 insight：2 个 70% / 1 个 30% 成功；失败返还+1博闻。
- 跨类拗转循环：蛾→灯→铸→刃→冬→心→杯→蛾；任意属性可升"启 Knock"；秘史不能被拗转。
- 降级：高级秘传+理性 → 两张低 2 级（100% 成功）。

## 8. 猎人与证据系统

- 疑心时节 + 场上有**秘氛 Mystique / 恶名 Notoriety**（及尸体）→ 生成猎人。
- 证据两级：初步证据 → 确凿证据 → 手下被审判/玩家被审判（有"当局人情"卡可免罪，否则 70% 失去手下或直接结局）。
- 对策成功率（攻略2）：
  - 蛾属性手下销毁证据：70%（手下无等级 30%）。
  - 心属性手下"迷惑社会"消恶名。
  - 谈话感化猎人：灯秘传对理想主义者 / 冬秘传对冷峻者，51–91%；成功猎人变可怜疯子。
  - 下毒：70%（"坚韧"特性猎人 30%）。
  - 刃手下暗杀：等级 0/1-4/5-9/10+ → 10%/30%/70%/100%，失败手下死亡。
- ⚙ 提示规则：场上有秘氛/恶名 + 猎人在场 → 建议清除声名；出现"确凿证据"→ 紧急。
- 防吸技巧：按住卡拖拽状态不会被吸走，空格暂停后可松手。

## 9. 教团与野心（飞升路线）

- 建团：熟人 + 2 级秘传 → 解锁建团 verb。灯之教派对新手最友好。
- 野心三路线（由投入属性决定，2→3 级后锁定）：理性→启明(要囚徒)、激情→感受(要尸体/疯子/囚徒)、健康→权力(要资金)。
- 升级节点：1→2 入梦+对应秘传；2→3 工作槽+牡鹿之门道路+6级秘传；3→6 只能在野心时节，喂囚徒/尸体/资金，共需 3 次。
- 囚徒来源：总部+任意人放谈话槽 → 监禁（产邪名）；探索雇员也可监禁。
- 手下属性用途速查：蛾=消证据、刃=杀猎人/抓囚徒、心=消恶名、冬=低概率杀猎人/取尸、杯=抓囚徒。
- 最终胜利：野心 6 级 + 主属性秘传总计 36+（小胜）/更高，仪式（如日落仪式）+高级影响+先见者援手。

| 层 | 需求 | 主要收益 |
|---|---|---|
| 林地 Wood | 激情 + 蛾4/灯2/启2 | 蛾影响、2-4级秘史 |
| 纯白之门 White Door | 健康 + 灯/启/冬 4+ | 冬6、启6影响、4-6级秘史 |

| 蛛门 Spider's Door | 灯/启/杯 8+ + 囚徒 | 杯10、8级秘史 |
| 孔雀之门 Peacock's Door | 灯/启/刃 10+ + 开门工具(碎门之钥最优) | 飞升素材（15级影响，仅存 60s） |

## 11. 远征（藏宝地）

- 配置：最多 2 手下 + 2 资金；先派 1 雇员+1 资金侦察看清全部危险再决定。
- 四类障碍：环境(铸/灯克)、封印门(启/铸克)、守卫(刃克)、诅咒(心/灯克，失败附身诅咒需 10+ 心影响对抗)。
- 成功率按克制属性等级：10+ 100% / 5-9 70% / 1-4 30% / 0 0%。失败 70% 损失一名手下（同类伤 3 次死亡）。
- ⚙ 每次远征完成必得 1 邪名——远征后应提示处理声名。

## 12. 常见死法速查（⚙ 全部可做成警报）

| 死法 | 触发 | 预防 |
|---|---|---|
| No More 绝望 | 3 恐惧被吸满 | 常备满足；恐惧≥2 即预警 |
| Glory 发疯 | 3 痴迷被吸满 | 常备恐惧/一瞬追忆 |
| 饿死/失血 | 无资金→健康转饥饿 | 资金<3 预警；及时入梦治疗 |
| 衰老拖死 | 病痛超时 | 病痛出现即紧急提示 |
| 隔槛望日 | 确凿证据审判 | 证据出现即紧急；备"人情"卡 |
| 献祭超时 | Poppy 事件不献祭 | 事件 verb 出现即提示 |
| 对手抢先 | 长生者欲望先到 7 级 | 中后期注意 rival 进度 |
