Cultist Adviser · 密教军师  v0.9.2
====================================

A read-only floating adviser for Cultist Simulator.
Advice only, hands off — no screen reading, no input
simulation, no save editing.

------------------------------------------------
1. Getting started
------------------------------------------------
1. Make sure the game itself is installed and runs. The
   adviser reads localized names and recipe data from the
   game's own files.
2. Double-click CultistAdviser.exe. The first launch takes a
   few seconds to build cache files (next to the exe);
   later launches are fast.
3. The window stays on top by default and can be dragged
   anywhere. The empty space to the right of the game board
   is a good home for it.

If your antivirus flags the first launch: that is a common
false positive against PyInstaller-packed programs, not
malware. You can:
- add it to the trust list; or
- run from source: https://github.com/fivood/cultist-adviser ; or
- build your own exe with the repo's launcher.py.

------------------------------------------------
2. The interface
------------------------------------------------
Top bar:
  [Spoiler tier]  [Language]  [Review]  [Sound]  [On top]

  Sound: when checked, a system beep plays whenever a NEW
  urgent alert appears — you'll hear the crisis even with
  the game fullscreen.

  Three spoiler tiers:
    * Keeper  — survival alerts only (things that kill you);
                fateful choices stay yours
    * Counsel — the default. Guidance without foretelling
                the outcomes of choices
    * Revelation — the all-seeing eye: hidden piles, choice
                outcomes, bookshop stock laid bare

Below the top bar sits an always-visible season forecast:
the next season and its countdown (at Revelation tier, the
full order of the remaining seasons). The line turns red
when the next season arrives and you have no counter ready.

The main area is three resizable panes:
  1. Suggestions — three visual tiers:
        ⚠ urgent (red, bold) — death countdowns and lethal events
        ● advice            — things worth doing
        ○ intel             — background: forecasts, the deck
        Right-click any non-urgent suggestion to mute it
        (ticking numbers won't bring it back; a real board
        change will). A "N muted" strip appears at the
        bottom — click it to restore all. Urgent alerts
        cannot be muted.
  2. Actions in progress — seconds left per verb; lethal
        verbs highlighted red. Double-click a row for the
        "startable right now" recipe list, computed from the
        cards actually on your table, exact cards included.
  3. Board resources — a collapsible tree grouped by
        category, with a filter box and a "timed only"
        toggle. Timed cards show where their countdown
        leads (e.g. "Restlessness → Dread") so you can tell
        which timers to fear. Double-click any card for the
        card dialog: the full decay chain on top, then two
        tabs:
          * Uses          — what this card combines with,
                            in which verb, for what result
          * How to obtain — how to get one when it's not
                            on the table
        Cards marked blue are cooling down (fatigue,
        exhausted, recovering) and will restore themselves;
        no need to rush them.

Season prophecy (Revelation tier):
  The intel section lists the exact order of this cycle's
  remaining seasons with arrival estimates. The pile's
  stored order IS the draw order (the game shuffles only on
  refill), so "two more Despairs, in ~4 and ~5 minutes" is
  a fact, not a guess.

3. FAQ
------------------------------------------------
Q. It says the save can't be found / stuck on "waiting".
A. The save lives by default at
     %USERPROFILE%\AppData\LocalLow\Weather Factory\Cultist Simulator
   If your game or save lives elsewhere, set environment
   variables:
     CULTIST_GAME_DIR = the game's install folder
     CULTIST_SAVE_DIR = the save folder
   (System Properties -> Environment Variables, or set them
   temporarily in a terminal.)

Q. Card names show in the wrong language.
A. Switch the top-bar Language box. Note the names come
   from the game's own localization files — the adviser can
   show whichever language your game copy ships.

Q. The game updated and some names look off.
A. Delete the two json caches next to the exe
   (lexicon_cache.json, knowledge_cache.json); they rebuild
   on the next launch.

Q. Data doesn't refresh until I alt-tab.
A. Fixed in v0.4.0. Older versions relied on the file
   timestamp, which Windows refreshes lazily; the adviser
   now reads the save content directly.

Q. I paused the game with Space but countdowns keep ticking.
A. The adviser can't see the pause key; it infers a pause
   when an expected save fails to arrive, so the freeze
   lags by up to a few dozen seconds. Nothing to worry about.

Q. How does the Review work?
A. Click "Review" in the top bar. The adviser snapshots
   every save change, filed per character (the same run
   keeps one file across GUI restarts). The window shows:
     * Post-mortem — cause of death in plain words with
       prevention notes (green congratulations for a win).
       Despair/Visions/arrest/starvation losses also get a
       timeline reconstructed from this run's own data:
       when the threat piled up, how long the counter was
       missing, when the countdown began.
     * Event history — every key choice and event, timed
     * Action stats — how many times you did each action
     * Achievements — local unlock record, progress N/83,
       locked ones grouped by category; locked entries
       expand to how-to notes
     * A resource trend chart on top (Funds / Dread /
       Fascination)

Q. An achievement shows locked here but unlocked on Steam.
A. The adviser reads the local achievement file (also used
   by GOG and Steam-offline). If Steam's cloud record never
   reached the local file, launch the game once and Steam
   writes it down.

Q. Where are the save backups?
A. The adviser copies your save into a save_backups folder
   next to the exe: one routine copy every 2 minutes (newest
   30 kept), plus an immediate snapshot the moment a lethal
   countdown appears (filename tagged _danger, newest 10
   kept). To restore: close the game, rename a backup to
   save.json and copy it over the save folder. Disable with
   "backup": false in settings.json.

Q. How do updates work?
A. On launch the adviser asks GitHub once whether a newer
   release exists (nothing else is sent). If so, a banner
   appears under the top bar — click it to download and
   the adviser restarts on the new version by itself. To
   disable the check entirely, set "update_check": false
   in settings.json next to the exe.

Q. I'm lost and don't know what to do next.
A. Read the "Current arc" line in the suggestions — the
   adviser infers your main quest from the board (found the
   cult / dedication / Stag Door / Ambition seasons /
   endgame) and keeps it pinned, with "still missing"
   notes for whatever the next step lacks.

------------------------------------------------
4. Privacy & safety
------------------------------------------------
- The only network access is a once-per-launch version
  check against GitHub (disable with "update_check": false
  in settings.json). Nothing about you or your game is sent.
- Reads the save file only; never writes or edits it.
- Creates just two cache jsons and a settings.json next to
  the exe — delete them to reset.
- Run histories are jsonl files in the logs folder, free to
  delete or share; they contain nothing beyond game state.
